# Product Rating Analysis System

## Overview

The system reads customer and product rating data from a text file, stores them using:

- A custom **sorted Linked List** (by customer ID)
- A **Two-Dimensional Array** for product ratings

It performs statistical analysis and predicts missing ratings using a similarity-based algorithm.

## Key Features

- Sorted linked list insertion
- File parsing
- Similarity-based rating prediction
- Product average calculations
- Country and occupation-based filtering

## Technologies

- Java
- Object-Oriented Programming
- Custom Linked List implementation
- File I/O
- 2D Array Data Structure
- Similarity Based Algorithm
